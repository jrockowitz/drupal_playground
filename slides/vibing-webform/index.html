Elements

  • Textfield, Email, URL, Number, Tel, Password, Hidden.
  • Checkboxes, Radio buttons, Select, Combobox, Autocomplete.
  • Textarea, Markup, HTML, Processed text.
  • Details, Fieldset, Container, Columns, Tabs, Accordion.
  • Table, Table select, Image select, Signature pad.
  • File upload, Managed file.

Elements

  • Rating, Likert, Range, Color.
  • Date, Time, DateTime, Datelist, Week, Month, Year.
  • Terms of service, Same-as checkbox.
  • Composite and custom composite elements.
  • Computed elements, prepopulation, and conditional logic.
  • Validation, masks, help text, and custom element plugins.

Handlers

  • Email with Twig templating.
  • Remote post with JSON or form data.
  • Action, settings, debug, and log handlers.
  • Conditional handler triggers.
  • Custom handler plugin API.

Integrations

  • Salesforce, HubSpot, Mailchimp, Stripe, and Slack.
  • Google Sheets and Jira.
  • Entity generation, Views, Rules, and webhooks.

Submissions Management

  • Customizable admin tables and exports.
  • View, edit, delete, resend, flag, star, and lock.
  • Audit logs, draft autosave, and submission limits.
  • Confirmation pages, serial numbers, and anonymous links.
Open submissions

Access and Security

  • Role-based access per form and per submission.
  • Element-level access control.
  • CAPTCHA, reCAPTCHA, Honeypot, and Antibot support.
  • Scheduled open and close dates plus IP tracking.

Display and Theming

  • Multiple display modes and confirmation experiences.
  • Block placement and entity reference support.
  • Per-form CSS and JavaScript.
  • Theme hook suggestions and print-friendly views.
