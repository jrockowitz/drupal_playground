Version Callout

Example Module 2.x for Drupal 11

Version 2.x is a full rewrite targeting Drupal 11. It introduces a new configuration UI and drops support for Drupal 9. See the upgrade guide if you are migrating from 1.x.

Help Callout

Upgrading from 1.x to 2.x

Run drush updatedb after updating the module. Configuration keys have changed — review the migration guide before upgrading on production.
