# Webform Issue Maintenance Tracker

Last reviewed: 2026-06-12

This directory contains committed public notes for non-security Webform issue
queue work. Use it to track issues selected by the maintainer for local agent
work, review, test coverage, or follow-up.

Do not use this directory for security issues, private details, secrets, exploit
prose, or confidential Drupal.org/GitLab data. Use the Webform security workflow
for security-sensitive work.

## Indexes

- Public issue work index: [index.md](index.md)
- Individual issue notes: [issues/](issues/)

## Layout

```text
webform-issue-maintenance/
  README.md
  index.md
  issues/
    README.md
    _template.md
    <drupalorg-node-id>.md
```

## Update Order

1. Update or create the individual issue note.
2. Update [index.md](index.md).
3. Update this README only if the directory map or update order changes.
