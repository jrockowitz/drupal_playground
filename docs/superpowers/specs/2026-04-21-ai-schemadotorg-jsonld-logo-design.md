# AI Schema.org JSON-LD Logo Design

## Goal

Create a logo direction for the Drupal module `ai_schemadotorg_jsonld` that communicates Schema.org knowledge graph structure with an AI accent.

## Approved direction

- Visual style: literal/schema-themed
- Core motif: knowledge graph plus AI stars
- Composition: asymmetric graph cluster
- Container: no page or document shape
- Node count: 8 total round nodes
- Hierarchy: 1 larger central node with 7 smaller connected nodes
- Node shape: round circles only
- Accent: 2 small AI stars placed in open space near one upper quadrant

## Design intent

The symbol should read first as a connected knowledge graph, not as a page, script tag, or generic AI badge. The AI stars are secondary and should feel like an intelligent or generated enhancement to the graph rather than additional graph nodes.

The asymmetric layout should avoid looking like a rigid diagram. The central node remains the anchor, while the surrounding nodes use uneven spacing and edge lengths to create a more organic cluster. The mark should still remain compact enough to read well at Drupal.org avatar sizes.

## Constraints

- Keep the icon simple enough to remain legible at small sizes.
- Avoid a document silhouette or UI frame.
- Avoid turning the stars into extra graph nodes.
- Use clean vector-friendly geometry and consistent stroke logic.
- Keep the final composition centered with enough padding for square export formats.

## Recommended visual system

- Primary motif: connected circle nodes with thin, consistent connector lines
- Secondary motif: 2 AI sparkle stars
- Suggested balance: medium visual density, slightly compact cluster
- Tone: technical, clean, contributor-friendly, not playful

## Next implementation outputs

1. Create 3 polished image-generation prompts based on this direction.
2. Optionally generate 1-3 raster logo candidates for review.
3. If a candidate is approved, convert it into a clean SVG-ready implementation path.

## Notes

This spec records the approved design direction only. It does not finalize palette, stroke weight, or exact star geometry yet.
